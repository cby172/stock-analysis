中国核电（601985.SH）深度投资价值分析报告

一、公司概况

公司全称：中国核能电力股份有限公司

股票代码：601985.SH（上海证券交易所主板）

所属行业：电力、热力生产和供应业（核电运营）

企业性质：中央国有企业（中国核工业集团有限公司控股，实际控制人为国务院国资委）

公司类型：重资产型 + 公用事业型公司——核心资产为核电机组，以发电量x上网电价为收入来源，属于典型的资本密集型、长周期、高壁垒行业。

主营业务：核电站的建设、运营和管理；核电技术的研发与服务；核燃料循环相关业务。

A股上市日期：2015年6月10日 | 当前总股本：205.68亿股

1.1 行业定位与宏观角色

中国核电是中国大陆第二大核电运营商（仅次于中广核电力），也是A股最纯粹的核电投资标的。作为国家战略性能源基础设施运营商，核电在"双碳"目标中占据核心地位——核电是唯一可大规模替代火电的清洁基荷能源（24小时不间断供电），风光水电均存在间歇性或季节性限制。

根据行业预测，2030年中国核电装机目标约1.1亿千瓦、2035年1.5亿千瓦。当前在运约6100万千瓦，未来十年核电装机将翻倍增长，中国核电作为运营主体将直接受益。

1.2 核电行业核心特征

  • 极端高壁垒：核电运营牌照仅中核集团、中广核集团、国家电投三家持有，具备天然垄断属性
  • 超长产业链：核电站从核准到商运平均8-10年（FCD到并网5-6年+前期审批3-4年），建设周期远长于火电、风电
  • 成本结构特殊：折旧占总成本40-50%、燃料成本仅占20-25%（火电燃料占70%+），铀价波动影响远小于煤价
  • 收入端管制：核电上网电价由国家发改委核定（"一厂一价"或标杆电价），不同机组电价在0.37-0.43元/度之间
  • 现金流特征：投运前巨额资本开支->投运后稳定现金流（折旧为非现金支出，OCF远高于净利润）

二、商业模式分析

2.1 商业类型判定

中国核电属于"重资产运营型 + 公用事业型"的复合模式。其商业本质是：

核心商业公式：收入 = 装机容量（万千瓦）x 发电利用小时数（小时）x 上网电价（元/度）x 厂用电率折扣

利润来源：电价收入 - 折旧（最大成本项）- 燃料成本 - 运维费用 - 财务费用（高杠杆->高利息支出）

增长驱动：① 新机组投运（装机量增长）— 外延增长；② 存量机组利用率提升 — 内生增长；③ 折旧到期后利润释放 — 时间价值

2.2 产品体系

中国核电的"产品"相对单一——电力。但可从不同维度拆分：

  • 按机组技术路线：二代改进型（M310/CPR1000）、三代自主（华龙一号HPR1000）、四代先进堆（高温气冷堆HTGR、快堆等）
  • 按电力市场：计划电（基数电量，有保障电价）与市场电（竞争性交易电量，电价略低）
  • 按区域分布：秦山（浙江）、田湾（江苏）、福清（福建）、三门（浙江）、昌江（海南）、漳州（福建）等核电基地
  • 按服务延伸：核电技术服务输出（运维、培训、核燃料供应）、核能供暖/供汽等综合能源服务

2.3 营销策略与客户分析

核电产品（电力）具有高度同质化特征，不需要传统意义上的营销。销售费用极低（2025年仅1.0亿元，费率0.12%），反映了以下特征：

  • 客户单一：客户为国家电网/南方电网（100%客户集中度），不需要市场营销
  • 产品无差异化：电力产品完全同质，竞争维度主要在成本端
  • 定价受管制：上网电价由发改委核定，不存在"提价"策略

"客户"画像分析：终端用户是全国电力消费者。核电作为基荷电源的竞争优势：① 发电成本稳定（不受化石燃料价格波动影响）；② 零碳排放；③ 发电利用小时数高（年均7500-7900小时，远超风电2000h、光伏1300h）。

2.4 战略执行框架

  • 规模化扩张：每年核准6-8台新机组，2024年核准11台创新高；在建机组约25台、装机超2700万千瓦（全球第一），预计2026-2030年每年投运2-3台
  • 技术迭代：从二代->三代（华龙一号）->四代（高温气冷堆/快堆），安全性提升且经济性改善
  • 核电+综合能源：核能供暖（秦山核能供暖示范项目）、核能制氢、同位素生产等多元化布局
  • 走出去：华龙一号海外首堆（巴基斯坦K2/K3）已投运，阿根廷等项目持续推进

2.5 生产/供应链模式

  • 核燃料供应：前端由中核集团内部体系供应（铀矿开采->转化->浓缩->元件制造），中核拥有国内主要铀矿资源和完整的核燃料循环产业链
  • 设备制造：东方电气、上海电气、哈电集团为三大主机供应商；一重、二重提供核岛锻件
  • 建设施工：中国核建（中核集团旗下）承担核岛安装工程
  • 运营维护：大修周期12-18个月/次，每次约30-60天，大修期间机组停运->影响当年发电量

2.6 成本结构与规模效应

2025年成本拆分（营收821亿元）：

  • 营业成本：482亿元（占比58.7%），其中折旧约占217亿元（推测占比45%）、核燃料约106亿元、运维人工等约159亿元
  • 财务费用：68.3亿元（费率8.33%），核电高杠杆运营的必然结果
  • 管理费用：41.2亿元（费率5.02%）

核电存在显著的规模效应：① 同一基地多机组共享基础设施，边际建设成本递减；② 标准化设计（华龙一号批量化建造）降低单位造价；③ 大修资源共享降低运维成本。

2.7 技术供应链分析

中国核电技术已实现三代自主化（华龙一号完全自主知识产权），但部分关键设备仍存在对外依赖：核级仪控系统（DCS）、核级焊材、部分核级泵阀等。国产化率已从早期的30-40%提升至当前的85%+，供应链安全持续改善。

2.8 效率分析

  • 机组利用率：2024年核电机组平均利用小时数约7800小时（全球领先水平），容量因子约89%
  • 厂用电率：核电约5-6%（低于火电6-8%）
  • 大修工期控制：多次打破华龙一号大修工期纪录，大修管理水平持续提升
  • WANO指标：多台机组在WANO（世界核电运营者协会）综合指数排名全球前1/4

三、行业概述与竞争格局

3.1 中国电力行业全景

  • 总装机：2025年末全国发电装机容量约33.5亿千瓦，其中火电47%、太阳能22%、风电16%、水电12%、核电1.8%
  • 总发电量：2025年约9.6万亿度，核电发电量约4500亿度，占比约4.7%
  • 核电定位：虽占比不高，但作为唯一清洁基荷能源，战略地位远超装机占比

3.3 宏观逻辑

短期逻辑：2025-2026年动力煤价格回落至700-800元/吨区间，火电成本改善->市场化电价小幅回落，核电市场化电量可能受轻微挤压。但核电计划电价部分不受影响，整体盈利稳定性强。

中长期逻辑：① AI算力中心爆发式增长->全社会用电量增速有望维持5-6%；② "双碳"目标下火电新增受限->核电成为增量基荷主力；③ 核电审批常态化（连续4年核准8台以上），增长确定性高。

3.4 未来行业趋势预判

  • 审批加速：预计2030年前年均核准8-10台，远超2011-2019年（年均0-2台）
  • 技术代际升级：华龙一号->华龙一号改进型->四代堆梯次推进
  • 市场化深化：核电市场化电量占比从20%向40-50%推进
  • 核电+新能源融合：核电站配套风电/光伏项目，"核电+综合能源基地"模式兴起
  • 小堆(SMR)商业化：ACP100（玲龙一号）2026年投运，开启小型堆新市场

四、产品与服务深度分析

4.1 核心产品：电力

电力产品的特殊性：① 无法储存（需要瞬时平衡）；② 完全同质化（1度核电=1度火电=1度水电）；③ 需求刚性（用电量与经济高度相关但波动小）。

核电的经济性优势：

  • 成本竞争力：核电全生命周期度电成本约0.25-0.30元（含折旧），与煤电（0.25-0.40元取决于煤价）、光伏（0.20-0.30元但需配储）处于同一区间
  • 成本稳定性：铀燃料成本仅占20-25%，铀价翻倍->度电成本仅增加约0.03-0.04元（煤价翻倍->煤电成本增加0.15-0.20元）
  • 绿色溢价：在碳交易市场，核电零碳优势可折算度电溢价0.03-0.05元

4.2 机组资产分析

核电机组是一种特殊的"长期生息资产"：

  • 折旧年限：核岛部分25-35年、常规岛部分20-30年（实际使用寿命可达60年，经延寿可至80年）
  • 折旧到期效应：核电机组折旧到期后，成本骤降40-50%，利润将大幅跳跃式增长
  • 退役准备金：核电需计提退役准备金（计入成本），这部分现金在公司账上形成大量可使用资金

5.1.2 毛利率与净利率趋势

毛利率从2016年41.0%缓慢下降至2025年41.3%，累计下降-0.3个百分点。原因：① 三代核电机组（华龙一号）单位造价约1.6-1.7万元/千瓦，高于二代改进型1.2-1.3万元/千瓦->折旧负担更重；② 市场化电量扩大（电价略低于计划电价）。

净利率从2016年27.0%下降至2025年21.9%，降幅5.1个百分点。净利率下滑幅度大于毛利率，原因是财务费用持续高企（总资产负债率维持69%，每年利息支出约68亿）。

5.3.2 经营活动现金流

2025年OCF为374亿元，是归母净利（93.0亿）的4.0倍。这一"夸张"的倍数恰恰是核电商业模式的精华所在：

  • 折旧是非现金支出：每年计提大量折旧（估计约217亿），减少了账面利润，但不消耗现金
  • OCF是核电真正的"赚钱能力"：投资人看核电不能只看PE（受折旧扭曲），更要关注OCF（真实现金创造能力）
  • OCF稳定但增速温和：2025年OCF374亿，较2024年407亿下降8.1%，可能与市场化电量增加导致应收账款增加有关

5.3.3 投资活动现金流

2025年ICF为-947亿元（净流出），连续两年投资现金流净额接近947亿元——这是公司大规模建设新核电机组的直接体现。核电"烧钱建产能->几十年稳定收钱"的模式决定了：

  • 单台华龙一号造价约200亿，每台建设周期5-6年
  • 在建约25台机组意味着未来5-6年将持续投入约5000-6000亿元
  • 这些投入将在2030年后陆续转化为利润（投产后）

5.3.4 筹资活动现金流

2025年筹资现金流净流入530亿元，经营现金流（374亿）远不足以覆盖投资现金流（947亿），差额需通过债务融资和股权融资补充。这是核电运营商扩张期的标准财务特征。

5.3.5 自由现金流估算

粗略估计：维持性资本支出（存量机组技改+延寿）约142亿，扩张性资本支出（新机组建设）约805亿。

企业自由现金流（FCFF）≈ OCF - 维持性资本支出 = 374 - 142 = 232亿元。按当前市值1890亿计算，FCFF/市值 ≈ 12.3%，现金回报率有吸引力。

5.4 三表联动分析

三表交叉验证结论：中国核电的三大报表高度自洽，清晰地呈现了"重资产扩张型公用事业"的典型财务画像。

① 利润表显示"账面利润温和增长"——因为大量折旧吃掉利润；② 现金流量表显示"现金创造能力强劲"——OCF远超NP，是核心价值指标；③ 资产负债表显示"杠杆扩张中"——负债率近70%但持续下降，资产规模快速膨胀。

核心矛盾：报表显示公司处于"利润表难看、现金表好看"的阶段，这是重资产建设高峰期（在建工程资本化->折旧尚未开始）的典型过渡期。投资人需判断：新机组投产后，增量利润能否覆盖增量折旧+增量利息。

八、投资逻辑与风险分析

8.1 投资亮点（核心逻辑）

  1. 稀缺性：A股唯一纯核电运营标的，中广核电力仅H股上市，投资核电必须买601985
  2. 增长确定性：在建约25台机组（全球第一），2026-2035年装机翻倍增长路径清晰
  3. 现金流质量极高：OCF/NP≈4x，真实的现金创造能力被PE估值低估
  4. 折旧到期红利：早期机组逐步折旧到期->利润跳跃式增长
  5. "双碳"政策加持：核电是唯一可大规模建设的清洁基荷能源，政策长期鼓励
  6. 成本稳定：燃料成本占比仅20-25%，铀价波动影响远小于煤价对火电影响
  7. 利率下行受益：68亿财务费用对利率高度敏感，利率每降100bp->利润增厚约52亿

九、结论与投资建议

核心结论：

中国核电（601985）是中国A股中最具确定性的"长期生息资产"。公司集"高壁垒（核电牌照垄断）+ 高确定性（装机翻倍路径清晰）+ 高现金流（OCF/NP≈4x）+ 政策红利（双碳加持）"四大优势于一身。

当前20.3倍PE看似不便宜，但考虑到核电折旧扭曲了账面利润（OCF是NP的4倍），以及未来10年装机翻倍带来的增量空间，9.19元的股价具备中长期配置价值。PB1.61x处于历史中低区间，提供了一定的安全边际。

公司目前处于大规模资本开支高峰期（每年资本开支近947亿），这压制了ROE和EPS的短期表现，但也为2030年后的利润爆发埋下伏笔。投资者的核心判断是：是否愿意接受短期财务指标的平淡，以换取长期确定的增长。

中国广核（003816.SZ）深度投资价值分析报告

报告日期：2026年6月12日 | 最新收盘价：4.35元 | 数据范围：2015-2026年

深圳主板 中央国有企业 核电龙头 沪深300

一、公司概况

公司全称：中国广核电力股份有限公司

股票代码：003816.SZ（深圳主板） | 01816.HK（港股）

所属行业：电力行业（核电）——属于公用事业子行业

企业性质：中央国有企业——实际控制人为国务院国资委（通过中国广核集团有限公司控股约57%）

公司类型：产品型+服务型（电力生产+电网服务）——属于"出租型"商业模式变体：建造核电站→发电→售电给电网公司，赚取稳定电费收入。核心是重资产运营，类似长江电力的"水电站出租发电量"模式，核电则是"出租核电站发电能力"。

成立/上市：2014年3月25日成立 / 2019年8月26日A股上市（2014年12月10日H股上市）

总部：广东省深圳市

1.1 行业概述

核电行业是中国电力系统的重要基石之一，具有以下核心特征：

  • 高壁垒：核电运营需要国家核安全局颁发的《核安全许可证》，且需具备极高的技术能力、安全管理经验和资金实力。国内仅有中国广核、中国核电（中核集团）两家企业拥有核电运营牌照，形成"双寡头"格局
  • 强政策驱动：核电发展高度依赖于国家能源规划和政策——包括核电审批节奏（每台机组需国务院核准）、上网电价政策、核电中长期发展规划（2035年目标核电装机2亿千瓦）等
  • 现金流稳定：核电发电成本中燃料占比低（约15-20%，远低于火电的60-70%），且有长期购电协议保障消纳，经营现金流极为充沛
  • 重资产长周期：单台核电机组投资约150-200亿元，建设周期5-8年。在建工程体量巨大、折旧费用高、财务费用高

截至2026年初，中国在运核电装机约5,800万千瓦（58GW），在建约2,400万千瓦（24GW）。中国广核在运装机约3,200万千瓦，占国内核电市场份额约55%，是中国最大的核电运营商。

二、商业模式与产品力

2.1 业务结构

中国广核的核心业务为核电站的建设、运营与管理，主要收入来源为向电网公司销售电力。

  • 核电运营（营收占比约85-90%）：通过旗下控股核电站（大亚湾、岭澳、阳江、台山、防城港、红沿河等）发电并网销售
  • 核电建设及技术服务（营收占比约5-8%）：为在建核电机组提供工程管理和技术服务
  • 新能源业务（营收占比约5-8%）：集团层面的风电、光伏等新能源配套（上市公司体内占比较小）

定价权分析：核电上网电价由发改委制定标杆电价（约0.39-0.45元/度），部分电量参与市场化交易。核电具有一定定价权，但主要受政策约束而非市场定价。与茅台等消费品的定价权不同，核电的定价权体现在：① 发电成本低于标杆电价（毛利率约32-42%），有合理利润空间；② 作为清洁能源，优先上网保障；③ 核电运营许可证的天然排他性。但限制：① 电价调整需发改委审批；② 市场化交易比例提升（已达约35-40%）带来电价下行压力。

2.2 产品体系详细分析

核心产品：核电上网电量

  • 华龙一号（HPR1000）：中国自主三代核电技术，单机容量约120万千瓦。防城港3/4号机为首批示范工程，已商运。是未来增长的核心驱动力
  • EPR技术（台山核电站）：引进法国EPR技术，单机175万千瓦，为全球最大在运核电机组。但曾因设备问题多次停机，利用小时数受影响
  • CPR系列（大亚湾、岭澳、阳江、红沿河等）：中国改进型压水堆技术，装机容量约100-108万千瓦/台，占总装机约70%，运营成熟度最高

产品结构特征

  • 收入产品结构（2025年）：核电售电收入约87%（约658亿），核电建设服务约8%（约61亿），其他（新能源等）约5%（约38亿）
  • 毛利率差异：核电运营毛利率约35-42%（稳定），核电建设服务毛利率较低约15-20%
  • 核电机组利用率：历史平均约7,300-7,600小时/年（2019年前），2020年后降至约7,000-7,300小时
  • 上网电价走势：从2015年标杆电价约0.43元/度→2025年平均电价约0.38-0.41元/度（市场化占比提升）

2.3 客户/用户画像

  • 核心客户：国家电网、南方电网两大电网公司——100%上网电量销售给电网，典型的B2B模式
  • 客户集中度：极高。前五大客户（均为电网公司）营收占比超95%，其中单国家电网占比约50-60%
  • 客户粘性：极强——电网公司签有长期购电合同（5-10年），且核电享有优先上网地位
  • 地域分布：主要在广东（约占50%+）、福建、辽宁、广西等沿海省份

2.4 营销策略与执行

核电行业几乎不需要传统营销。销售费用极低——2025年仅0.4亿（占营收0.05%），主要是电力市场交易手续费。核电的"营销"核心在于：① 保障安全运营（最大的"营销"）；② 争取新建机组审批（决定未来增长）。在这两点上执行优秀——未发生二级及以上核安全事件，新机组审批自2022年起加速。

2.5 公司战略与执行验证

核心战略："做强核电主业 + 适度多元化发展"双轮驱动。

  • 装机增长验证：在运装机从2015年约1,600万千瓦→2025年约3,200万千瓦（翻倍），年均增长约7%——执行良好
  • 利用小时验证：从7,600小时→约7,100小时（下降约7%），受换料大修延长和电网调峰影响
  • 海外进展：英国Bradwell B项目仍在前期，海外战略执行慢于预期

2.6 生产/运营模式与效率分析

  • 生产模式：核燃料裂变→热能→蒸汽→汽轮机发电→并网。运营周期18个月（含换料大修2-3个月）
  • 成本结构（2025年）：总成本约617亿。其中：燃料100-120亿（16-19%），折旧180-200亿（29-32%），人员95亿（15%），财务费45亿（7%），维修80-90亿（13-15%），其他50-60亿（8-10%）
  • 规模效应：明显——管理费仅从24.1亿→28.6亿（+19%），远低于营收增速（+182%）
  • 经营现金流/营收比：约0.40-0.50——运营效率高，现金回收能力强

2.7 商业模式风险

  • 核安全风险（低概率高影响）：一次重大核事故可导致公司倒闭、行业停滞
  • 政策风险：审批节奏高度受政策影响
  • 电价风险：市场化交易占比上升，电价存在下行风险
  • 建设风险：新电站投资150-200亿/台，建设周期5-8年，存在超支延期风险
  • 利率风险：有息负债约1,800亿，利率每升1%则年化财务费用增18亿（减利约13-15亿）

三、关键财务指标分析（2015-2025年）

总览：中国广核近十年处于稳步增长阶段——营收从268亿增至757亿（+182%），归母净利润从67亿增至108亿（+61%）。利润增速低于营收增速的主要原因：① 折旧和财务费用随装机增长而大幅增加；② 电价下行；③ 新机组早期利用小时偏低。毛利率从46.8%降至32.0%，净利率从30.0%降至19.5%。

四、财务三表深度分析

4.1 利润表深度分析

  • 收入端：核电售电收入为核心（约85-90%），增长主要靠新增核电机组投产。2015-2021年营收年均增速约20%；2022-2025年增速放缓（装机青黄不接期叠加电价下行）
  • 成本端：毛利率从2015年46.8%降至2025年32.0%。原因：① 折旧增加（新机组转固）；② 维修费用上升；③ 边际成本较高机组投产拉低平均毛利率
  • 费用端：管理费+销售费+财务费合计约74.2亿（2025年），费用率约9.8%（费用/营收）。其中财务费45.2亿占比最高（61%），反映高杠杆特征
  • 利润端：归母净利润率约19.5%（2025年），处于公用事业合理区间（15-25%）。净利润10年CAGR约4.9%，增速不快但稳定

4.2 资产负债表深度分析

  • 资产结构：固定资产2,524亿（50%）+在建工程约1,200亿（24%）为核心资产，无商誉
  • 负债结构：负债率65.2%，以长期借款为主。有息负债约1,800亿，有效利率约2.5%（央企低融资成本优势）
  • 应收账款：约200-250亿（电网公司电费），因电网公司AAA信用评级，坏账风险极低
  • 偿债能力：经营现金流/有息负债≈16.7%，覆盖倍数偏低但稳定可持续

4.3 现金流量表深度分析

  • 经营活动：299-375亿/年，经营现金流/净利润比约2.8-3.5倍——因折旧和摊销是非现金支出，真实盈利能力远强于账面利润
  • 投资活动：持续大额流出（-117~-324亿/年），2024-2025年因防城港、惠州项目集中投入期而大幅增加
  • 筹资活动：正负交替，根据投资需求灵活融资。2020-2023年净偿还约175-244亿，2025年因加大投资而转为净融资

4.4 三表联动分析与商业逻辑验证

三表逻辑闭环验证：利润表显示营收和利润稳定增长→经营现金流验证了盈利质量（现金流远超净利润）→资产负债表显示稳健扩张（资产增长、负债率下降）。三者共同验证了"重资产但现金流极佳"的优质公用事业商业模式。

经营现金流（约300亿）> 归母净利润（约108亿）+ 折旧（约180亿）——核电站设计寿命40年（延寿可达60年），会计折旧完成后自由现金流将大幅跃升。

展望：若能保持每年核准2-3台新机组→装机持续增长→营收年均增5-8%→经营现金流稳定在300亿以上。随着大亚湾等早期机组在2030-2040年间折旧完成，自由现金流有望显著提升。

4.5 杜邦分析

  • ROE = 净利率 × 资产周转率 × 权益乘数
  • 2025年：ROE ≈ 9.1% = 净利率19.5% × 资产周转率0.15 × 权益乘数2.87
  • 净利率（19.5%）：公用事业正常水平
  • 资产周转率（0.15）：极低——重资产行业特征，这是核电ROE不高的根本原因
  • 权益乘数（2.87）：较高（负债率65%），反映了高杠杆运营特征
  • 核心矛盾：低周转率是制约ROE提升的关键。提升路径：① 提高利用小时数；② 优化资本结构降息；③ 华龙一号效率优化

5.3 DCF估值（现金流折现法）

核心假设：基于2025年经营现金流299.7亿为基数，预计未来10年自由现金流年均增长3%（装机增量+电价缓降），折现率8%（央企+现金流确定性强），终值增长率2%。

  • 简化DCF模型（经营现金流直接折现）：Q = FCF x (1+g) / (r-g) = 300 x 1.03 / (0.08-0.02) = 5,150亿
  • 每股DCF估值 = 5,150亿 / 504.99亿股 = 10.2元/股
  • 考虑每股净负债（有息负债-现金）/股约3.0元，调整后股权价值约7.2元/股

敏感性分析：若折现率7%、增长2%→每股价值8.5元；若增长0%→每股价值5.5元。当前股价4.35元，DCF测算处于合理偏低估区间。

六、投资逻辑

6.1 投资亮点

  1. 垄断性行业壁垒：国内仅中广核、中核两家拥有核电运营牌照，双寡头格局稳固
  2. 经营现金流极强：300亿+/年，是净利润的3倍。折旧完成后自由现金流将大幅跃升
  3. 央企信用背书：AAA信用等级，融资成本低（有效利率约2.5%）
  4. 政策确定性增强：2022年起核电审批加速（年核准6-8台常态化），未来增长路径明确
  5. 清洁能源转型受益者：核电基荷电源地位不可替代
  6. 高分红潜力：分红率持续提升（从约34%→38%），经营现金流支撑下有望进一步提高

6.2 主要风险

  1. 核安全风险（低概率高影响）：一次重大核事故可导致公司面临生存危机
  2. 电价市场化下行风险：加权平均上网电价从约0.42→0.39元/度，若持续下降将侵蚀利润
  3. 审批不及预期风险：若核电审批放缓，装机增长目标无法实现
  4. 财务费用压力：有息负债约1,800亿，利率每升1%则年化减利约13-15亿
  5. 机组利用率下降：新能源并网后调峰需求增加，核电利用小时数有下降趋势

6.3 需要关注的关键变化

  • 核电审批数量：每年核准机组数——若回落至4台以下则为负面信号
  • 平均上网电价趋势：市场化交易比例和价格——影响营收方向
  • 新机组投产进度：防城港4号、惠州太平岭1/2号等——在建转固时间决定营收增量
  • 利率走势：因高负债率，利率下行有利（节省财务费），上行不利
  • 分红率变化：若提升至50%+，股息率将升至3%+，成为重要估值催化剂
  • 延寿政策：大亚湾等早期机组若获批延寿至60年，将大幅提升存量资产价值

七、宏观与行业分析

行业龙头地位：中国广核是国内最大、全球第三大的核电运营商（仅次于法国EDF、韩国KHNP）。在运装机约3,200万千瓦（国内核电市占率约55%），与中核集团形成双寡头格局。

宏观逻辑：

  • 碳中和战略利好：核电是零碳基荷电源，在双碳目标下具有不可替代性
  • 电力需求增长：全社会用电量年均增速约4-6%（AI算力、新能源汽车等拉动），核电需同步增长
  • 能源安全战略：核电减少对进口能源依赖，地缘政治紧张时期战略价值凸显
  • 低利率环境利好：高杠杆行业（负债率65%），低利率降低财务费用

政策环境：

  • 支持性：国务院《核电中长期发展规划》明确2035年核电装机目标2亿千瓦（约为当前3.5倍）。2022年以来核准节奏明显加速（年6-8台），政策转向积极
  • 限制性：电力市场化改革对核电电价形成下行压力。核燃料后处理基金（0.026元/度）等附加费用也在增加

央企治理：中国广核为中央国有企业，实际控制人为国务院国资委（通过中国广核集团间接控股约57%）。管理层稳定、经营目标以安全为首。央企身份有利于项目审批沟通，但股权激励较民企仍有限（2023年推出首批股权激励，覆盖约200名骨干）。

未来行业趋势：

  • 装机规模持续增长：2035年目标2亿千瓦，未来10年增长空间约2.5倍
  • 技术升级：华龙一号造价从约1.8→1.6万元/千瓦以下，经济性提升。第四代高温气冷堆、SMR等新技术研发中
  • 市场化改革深化：电力现货市场完善，核电基荷价值通过容量电价机制得到更充分体现
  • 延寿与退役：大亚湾运行超30年，若获批延寿至60年将提升存量价值。退役基金累计约150-200亿，责任可控

八、总结与投资建议

核心结论：中国广核是中国核电行业的双寡头龙头之一，拥有核电站运营的天然垄断壁垒。经营现金流极为充沛（300亿+/年，净利润的3倍），在国家双碳目标和能源安全战略下增长路径明确。当前PE 22.5x处于历史偏低分位，具备较好的安全边际。

但ROE仅约9%且提升有限（重资产运营的本质），利润增速较低（CAGR约5%），属于"稳但慢"的投资品种——类似长江电力的现金牛逻辑，适合追求稳定股息和低波动的保守型投资者。

风险收益比：若核电审批持续加速+电价稳定，PE有望修复至15-18x（对应3.4-4.1元，上涨空间约0-10%）。若电价下行或审批放缓，PE可能回落至12-15x（对应2.7-3.4元，下行风险约10-20%）。当前股价4.35元处于合理偏低区间。
